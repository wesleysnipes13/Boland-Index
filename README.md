The Boland Index

Status: Draft / Unvalidated Version: 1.0

The Boland Index is an unvalidated scoring system designed for otherwise healthy people to quantify analyze and identify areas for improvement within their life. It assesses five key pillars that I believe set the foundation for longevity and health. These five pillars are evidence-based and correlate well with all-cause mortality reduction and cognitive resilience.

The 5 Pillars

The index weighs the following five domains equally (10 questions each):

Nutrition

Movement

Sleep

Social Connection

Purpose

Scoring Methodology

The Index uses a Likert Scale (1-5) for 50 items.

1 (Strongly Disagree): Low adherence / High risk behavior.

5 (Strongly Agree): Optimal longevity behavior based on current preventative medicine literature.

Total Possible Score: 250 Pillar Max Score: 50

Interpretation

Asymmetry: Significant variance between pillars (e.g., high Movement, low Social) indicates opportunities to focus and improve that health span profile.

< 30 per pillar: Significant area for intervention.

45 per pillar: Optimal adherence.

Disclaimer

This tool is for informational purposes only. The Boland Index is an unverified, unvalidated metric. It has not been subjected to clinical trials or peer review. It is not a diagnostic tool and does not constitute medical advice. All users should consult with a qualified healthcare provider regarding their specific health needs.

Contributing

Pull requests to refine question phrasing or update questions based on new literature (e.g., specific protein thresholds, zone training protocols) are welcome.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
